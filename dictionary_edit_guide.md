# 绿县文旅词库可编辑文档说明

## 词库文件位置

词库文件存储在: `lv_trlator/resources/dictionaries/`

## JSON词库格式

每个词库文件采用JSON格式，包含多个分类(section)。每个词条包含以下字段：

```json
{
  "category_name": [
    {
      "id": "唯一标识符",
      "zh": "中文词条",
      "hani": "哈尼语词条(拉丁字母)",
      "pronunciation": "中文发音参考(可选)",
      "category": "细分分类(可选)",
      "intro": "词条介绍(可选)"
    }
  ]
}
```

## 词库文件列表

| 文件名 | 分类 | 说明 |
|--------|------|------|
| daily.json | 日常用语 | 问候、家庭、称谓等 |
| lvyou.json | 旅游词汇 | 景点、交通、住宿等 |
| festival.json | 节日词汇 | 节日、民俗、活动等 |
| food.json | 美食词汇 | 菜名、食材等 |
| building.json | 建筑词汇 | 建筑名称、部件等 |
| clothing.json | 服饰词汇 | 服装、饰品等 |
| etiquette.json | 礼仪词汇 | 礼仪用语等 |
| direction.json | 问路词汇 | 方向、距离等 |

## 添加新词条步骤

### 方法1：直接编辑JSON文件

1. 用文本编辑器打开对应的JSON文件（如daily.json）
2. 在对应分类的列表中添加新词条
3. 保持JSON格式正确（注意逗号、引号等）
4. 保存文件后重启应用

### 方法2：导出为Excel编辑

1. 使用在线工具将JSON转换为Excel
2. 在Excel中编辑词条
3. 转换回JSON格式
4. 替换原文件

## 词条ID编码规范

词条ID采用分类前缀+序号的方式：
- d开头：daily日常用语
- ly开头：lvyou旅游词汇
- fj开头：festival节日
- fd开头：food美食
- bd开头：building建筑
- cl开头：clothing服饰
- et开头：etiquette礼仪
- dr开头：direction问路

## 测试词条

运行以下命令初始化测试词条：

```bash
python init_dictionaries.py
```

这将自动生成所有词库文件。

## 注意事项

1. **编码格式**：所有文件必须使用UTF-8编码
2. **JSON格式**：确保JSON格式正确，可使用JSON校验工具验证
3. **哈尼语准确性**：新增词条建议咨询懂哈尼语的人士确保准确性
4. **文化尊重**：确保词条翻译符合哈尼族文化习惯，无贬义或歪曲
